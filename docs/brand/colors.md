# Nexora IA — Brand Colors

| Nombre        | Hex       | Uso                                      |
|---------------|-----------|------------------------------------------|
| Obsidiana     | `#0D0F0E` | Fondo principal                          |
| Esmeralda     | `#1A3C2E` | Bordes, fondos secundarios, separadores  |
| Lima Eléctrico| `#A8FF3E` | Acento principal, CTAs, íconos activos   |
| Blanco Cálido | `#F2F0EB` | Texto principal                          |
| Gris Perla    | `#C8C9C0` | Texto secundario, subtítulos             |

## Combinaciones aprobadas
- Fondo Obsidiana + texto Blanco Cálido + acento Lima ✅
- Fondo Esmeralda + texto Blanco Cálido ✅
- Lima sobre Obsidiana (CTAs, botones) ✅

## Fuente
- **Display / Headings:** Inter 700 o Space Grotesk 700
- **Body:** Inter 400 / 500
- **Wordmark:** Inter 700, letter-spacing 3px (NEXORA) | Inter 600, letter-spacing 9px (IA)
